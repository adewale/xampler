# KV 02 — Binding

Pythonic KV helpers for text, JSON, existence checks, deletes, and key listing.

Routes: `PUT /text/<key>`, `GET /text/<key>`, `PUT /json/<key>`, `GET /json/<key>`, `DELETE /keys/<key>`, `GET /keys?prefix=...`.
