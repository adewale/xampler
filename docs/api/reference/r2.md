# R2

## Import

```python
from xampler.r2 import R2Bucket, R2Range, R2HttpMetadata
```

## Copy this API

```python
bucket = R2Bucket(env.BUCKET)
await bucket.object("notes/a.txt").write_text("hello")
text = await bucket.object("notes/a.txt").read_text()
```
